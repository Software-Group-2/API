# API

We will use Node.js Express with MongoDB Architecture to make login and signup


## requirements

- Express 4.17.1
- cookie-session 1.4.0
- bcryptjs 2.4.3
- jsonwebtoken 8.5.1
- mongoose 5.13.13
- MongoDB

# Initialize setup


```shell
npm install  --save
```

```shell
node server.js
```

open your bowser with url http://localhost:8080/
